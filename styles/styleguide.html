Typography

Applied directly to h1–h3, p and li — no classes needed for basic text.

Header XL / H1 — 40px bold

Header LG / H2 — 24px bold

Header SM / H3 — 18px bold

Body MD / p — 14px regular. This is the default paragraph style, used for all body copy.

Body SM — 12px, used for captions and tags (e.g. .attribute-list).

--font-size-header-xl is used manually on .content-homepage h1 only — every other h1 uses --font-size-header-lg.
